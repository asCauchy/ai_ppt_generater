## 1. 缓动曲线（选最接近的）

**全局缓动哲学：**
- [ ] ease-out — 元素快速进入，缓慢停稳（最常见，Apple 风格）
- [ ] ease-in-out — 起止都缓，中间快
- [ ] spring — 弹性回弹，有 bounce 感（Stripe 风格）
- [ ] linear — 匀速，没有加速减速（编辑/纪实风格）
- [ ] custom — 自定义，有明显个性

## 4. 场景转场

**幻灯片之间怎么切换：**
- [ ] 溶解(fade) — 缓慢淡入淡出
- [ ] 硬切(cut) — 瞬间切换
- [ ] 推(push) — 新页从左/右推入
- [ ] 缩放(zoom) — 放大进入或缩小退出
- [ ] 变形(morph) — 元素形状连续变化到下一页
- [ ] 颜色偏移 — 背景色先变，内容后变

**转场时长：**
