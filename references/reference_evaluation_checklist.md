# Reference Evaluation Checklist

Score each dimension 1–5 (1 = absent/broken, 5 = exemplary). A reference scoring above 35/50 is likely worth distilling. Below 25 is noise.

## 1. Motion Coherence
Does movement feel like a single authored gesture rather than a collection of independent animations?
- [ ] Elements move as an ensemble, not as isolated actors
- [ ] Stagger delays feel intentional (orchestrated), not random
- [ ] No competing motion paths that confuse the eye
- **Score (1–5):** ___

## 2. Spatial Continuity
Does the frame respect consistent spatial rules across the sequence?
- [ ] Elements travel along predictable, logical paths
- [ ] Scale changes feel grounded — objects recede or advance in space
- [ ] Screen-to-screen transitions preserve element positions when appropriate
- **Score (1–5):** ___

## 3. Camera Language
Is the virtual camera used deliberately to frame, reveal, or emphasize?
- [ ] Camera moves (push, pull, pan, rack focus) serve narrative purpose
- [ ] Depth of field / parallax is intentional, not gimmicky
- [ ] Camera behavior matches the content's emotional register
- **Score (1–5):** ___

## 4. Typography Hierarchy
Does type carry the information architecture through scale, weight, and position?
- [ ] Clear size/weight/luminance hierarchy — no ambiguity about reading order
- [ ] Type animations respect readability (no orphan words, no mid-word breaks)
- [ ] Font pairings create meaningful contrast, not visual noise
- **Score (1–5):** ___

## 5. Negative Space
Is emptiness used as an active design element?
- [ ] Breathing room around key elements is intentional, not leftover
- [ ] Dense information clusters are balanced by open zones
- [ ] Negative space guides focus rather than feeling sparse or abandoned
- **Score (1–5):** ___

## 6. Emotional Pacing
Does the temporal rhythm match the content's emotional weight?
- [ ] Pacing accelerates for energy, decelerates for gravity — deliberately
- [ ] Holds and pauses are used to let information land
- [ ] No section feels rushed or dragged beyond its natural duration
- **Score (1–5):** ___

## 7. Cinematic Rhythm
Does the sequence have a discernible beat structure — fast/slow/fast, build/release?
- [ ] The piece has a clear act structure (setup → development → peak → resolution)
- [ ] Intensity varies across the timeline — no flatline energy
- [ ] Rhythm feels composed, not algorithmic
- **Score (1–5):** ___

## 8. Transition Intelligence
Do transitions carry semantic meaning rather than just moving between screens?
- [ ] Transition type is chosen for content, not variety (match-cut for continuity, hard cut for contrast, dissolve for passage)
- [ ] Elements bridge across transitions when continuity matters
- [ ] No "transition for transition's sake" — every shift earns its place
- **Score (1–5):** ___

## 9. Visual Restraint
Does the composition do less than it could — and more because of it?
- [ ] No gratuitous effects, particles, glows, or decorative motion
- [ ] Color palette is disciplined — accent color has a job
- [ ] Every visible element is load-bearing for the message
- **Score (1–5):** ___

## 10. Focus Guidance
Does the motion direct the viewer's eye to exactly what matters, exactly when it matters?
- [ ] Entry order and motion path create a clear attention sequence
- [ ] No competing elements fighting for the eye at the same moment
- [ ] You know where to look at every frame — no visual ambiguity
- **Score (1–5):** ___

---

**Total Score:** ___ / 50

**Verdict:**
- 40–50: Exemplar — full distillation recommended
- 30–39: Strong — distill specific dimensions where it excels
- 20–29: Marginal — file for pattern reference only
- Below 20: Noise — do not collect
