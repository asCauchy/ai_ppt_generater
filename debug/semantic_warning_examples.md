## Example 1: N02 — Conflict Collapse (run_02, slide 4)

**What the validator found:**

The slide declares `narrative_role: "conflict"` but content is structured as a balanced pro/con list (3 breakthroughs + 3 challenges + resolution sentence). The dramatic function of "conflict" (sustaining tension, creating stakes) is not realized.

```json
{
  "layer": "semantic",
  "rule": "N02",
  "severity": "warning",
  "message": "Conflict slide is a balanced pro/con list — lacks genuine tension",
  "location": "$.slides[4]",
  "context": {
    "narrative_role": "conflict",
    "issue": "balanced_procon_structure",
    "content_excerpt": "突破：百度文心一言...突破：华为昇腾芯片...突破：中国AI论文数量全球第一...挑战：高端芯片制造仍受制于光刻机...挑战：基础软件生态薄弱...挑战：AI顶尖人才仍存在外流...正视短板，才能更好地发挥优势..."
  }
}
```

**Why this matters:** When conflict is neutralized into pros-and-cons, the audience experiences no dramatic tension. The emotional arc (curious → impressed → engaged → SURPRISED/conflict → inspired → hopeful) is broken — there's no real low point for the vision slide to rescue.

**Suggested fix:** Restructure to make tension the dominant frame. Lead with stakes ("If China loses chip access, the AI ecosystem faces 3-5 year regression"). Separate breakthroughs to the evidence slide. Let conflict be conflict.

## Example 2: N05 — Vision Drift (run_02, slide 5)

**What the validator found:**

The slide declares `narrative_role: "vision"` but has 0 future-oriented dimensions. Content describes current-state quantitative data (6-dimension radar chart with China scores 75-95 vs global average).

```json
{
  "layer": "semantic",
  "rule": "N05",
  "severity": "warning",
  "message": "Vision slide has only 0 future dimension(s) — needs >=2",
  "location": "$.slides[5]",
  "context": {
    "narrative_role": "vision",
    "detected_dimensions": {
      "future": false,
      "possibility": false,
      "long_term": false
    },
    "content_excerpt": "中国AI在论文、专利、人才储备上已位居世界前列，生态协同效应正在加速。"
  }
}
```

**Why this matters:** A "vision" slide that only describes the present isn't a vision — it's a status report. The audience doesn't see what's ahead. The presentation arc misses its third-act lift.

**Suggested fix:** Frame around "Next Decade" language: "By 2035, China's AI ecosystem will..." with specific future milestones, not just current rankings.

## Example 3: E01 — Emotional Content Mismatch (run_01, slide 3)

**What the validator found:**

The slide declares `emotional_role: "reflective"` but content has no reflective keywords (回顾/总结/思考/意味着/本质上), and instead uses confident/assertive language about ecosystem completeness.

```json
{
  "layer": "semantic",
  "rule": "E01",
  "severity": "warning",
  "message": "emotional_role='reflective' conflicts with content tone: no reflective keywords found in content",
  "location": "$.slides[3]",
  "context": {
    "emotional_role": "reflective",
    "mismatch": "no reflective keywords found in content",
    "content_excerpt": "从芯片设计（华为昇腾、寒武纪）到云计算（阿里云、华为云），中国拥有自主可控的算力底座..."
  }
}
```

**Why this matters:** When `emotional_role` doesn't match actual content tone, the speaker notes and emotional design cues will misdirect the presenter. The slide's visual design (color, imagery) may be tuned for "reflective" when the content is actually "confident/assertive".

**Suggested fix:** Either change emotional_role to "confident" or "convinced", or rewrite content to include reflection-triggering language ("What does this ecosystem completeness really mean for the next decade?").

## Example 4: E02 — Dual Emotional System Conflict (run_02, slide 4)

**What the validator found:**

The NarrativePlanner set `rhythm.emotional_state: "surprised"` for slide 4, but ContentWriter independently set `emotional_role: "skeptical"`. These are contradictory — surprise expects wonder, skepticism expects doubt.

```json
{
  "layer": "semantic",
  "rule": "E02",
  "severity": "warning",
  "message": "emotional_role='skeptical' contradicts rhythm.emotional_state='surprised'",
  "location": "$.slides[4]",
  "context": {
    "emotional_role": "skeptical",
    "rhythm_emotional_state": "surprised",
    "relationship": "contradiction"
  }
}
```

**Why this matters:** This is the most common and impactful pattern — two agents setting contradictory emotional signals because there's no shared emotional contract. The audience receives mixed signals: the rhythm says "wow moment", the content says "question this."

**Suggested fix:** Either align CW's emotional_role to NP's emotional_state, or formalize a precedence rule (NP's emotional_state binds CW).

## Example 5: N07 — Insight Flattening (run_03, slide 4)

**What the validator found:**

The slide has `narrative_role: "insight"` but matches 0/1 required pattern groups (reframing, deeper meaning, new perspective). Instead, content enumerates 6 factual achievements.

```json
{
  "layer": "semantic",
  "rule": "N07",
  "severity": "suggestion",
  "message": "narrative_role='insight' content matches only 0/1 required pattern groups",
  "location": "$.slides[4]",
  "context": {
    "narrative_role": "insight",
    "matched_groups": [],
    "required_groups": ["reframing", "deeper_meaning", "new_perspective"],
    "content_excerpt": "字节跳动TikTok推荐算法、大疆无人机AI视觉系统，已成为全球标杆产品。商汤、旷视等企业参与制定ISO/IEC国际AI标准超30项..."
  }
}
```

**Why this matters:** Insight slides should reframe the audience's perspective ("It's not about technology, it's about ecosystem dynamics"). Instead, they become "more facts" slides — indistinguishable from evidence slides.

**Suggested fix:** Add a reframing statement: "These aren't just export numbers — they signal a fundamental shift in who writes the rules of global AI."

## Cross-Run Pattern: `emotional_role='curious'` with no curious content

This pattern appears in **ALL 5 runs** on slide 0 (the cover/hook slide). The slide declares `emotional_role: "curious"` but the lead text is typically an assertive statement ("中国AI已不再是跟随者..."), not a curiosity-triggering question.

The validator catches this consistently:

```
[E01] emotional_role='curious' conflicts with content tone: no curious keywords found in content
```

**Root cause:** ContentWriter sets emotional_role to match the hook narrative function, but writes declarative lead sentences. A genuine "curious" opening should include rhetorical questions, mystery, or unresolved tension.
